# Learnloom launch film — Remotion

The revised launch film is a Remotion composition. It uses the real Learnloom
lesson capture for the product reveal, while the autonomous source-learning
scene is an illustrative visual metaphor for the intended user experience.

From this directory:

```sh
npm install
npm run studio
npm run render
```

The composition is `LearnloomLaunch` in `src/LearnloomLaunch.jsx`. The rendered
video is written to `../output/learnloom-launch-remotion.mp4`.
